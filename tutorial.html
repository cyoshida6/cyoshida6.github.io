1. Introduction¶

Bladee (pronounced "blade") is a Swedish rapper, singer, songwriter, and record producer, among other endeavors. His music, known to be industrial, etherial, experimental, and dark, has a massive cult following and has been known to not fit cleanly into the modern music industry's definitions of genre. His musical influences are known to vary greatly, ranging from Chief Keef to The Beach Boys, and his auto-tuned, emotional, and contemporary sound manifests itself in an inconceivable number of ways throughout his catalog. His unprecedented music has garnered a dedicated core fanbase and widespread familiarity for some of his more popular songs. As is normal for any artist, his catalog has tracks of varying degrees of popularity. Because of his variety in both style and popularity, in this report, we will explore the relationship between these factors.

2. Data Collection¶

2.1 Rationale¶

We will be comparing the songs in Bladee's discography on an individual basis. In order to quantify a song's musical style and popularity, we will use data from existing sources.

To find a song's popularity, we will look at Spotify's "popularity" value, showing the song's popularity on Spotify on a scale of 0 to 100. We will assume that a song's popularity on Spotify represents its popularity overall. Since Bladee's discography is available on all streaming platforms, we will assume that there is no discrepancy between Spotify and SoundCloud, for example, which is known for hosting content that cannot exist on other platforms for copyright reasons. We will also assume that there is no significant difference in users' preferences across all streaming platforms, such as Apple Music, for convenience. We will not look at music charts or awards because Bladee is not popular enough for that data to have any significance.

To find a song's musical style, we will look at Genius's tags for each song. Each song has about 0-10 tags, a phrase describing the genre or stylistic elements of a song. All of Genius' data is crowdsourced and peer-reviewed. We will assume that all data is accurate and that a similar level of effort and description was applied to each song.

    url = 'https://accounts.spotify.com/api/token'

2.3 API Calls and Web Scraping¶

We will use Spotify's catalog as reference for Bladee's discography. While the releases present on Spotify do not always represent an artist's complete catalog, we have found that this is not the case for Bladee. We will use Spotify's "album_type" classifier to find all releases classified as an 'album'. We will not consider singles or EPs as these types of releases are not always considered part of an artist's discography and often lack important context necessary to consider them fully realized releases. For example, considering a single outside of the context of an upcoming album it will be featured on does not paint a complete picture. We will also omit the deluxe version of the album "Good Luck" because the regular version is also present and the deluxe version contains remixes by other artists and producers, and thus may not be representative of Bladee and Mechatok's creative vision. (Mechatok is a producer on "Good Luck") We will consider songs that Bladee is not featured on but are on albums for which Bladee is a primary artist as they can still be considered a part of his discography.

import numpy as np
import pandas as pd
from bs4 import BeautifulSoup

# genius_get_tags gets song tags by scraping provided genius.com url
def genius_get_tags(url):
    return [x.get_text() for x in BeautifulSoup(requests.get(url).content).find('div', {'class': 'SongTags__Container-xixwg3-1 bZsZHM'}).findAll('a')]

# Get tracks for each album in spotify_album_ids
# Use API calls to find column values and fill DataFrame
def scrape(album_ids):
    df = pd.DataFrame(columns=['spotify_id', 'name', 'artists', 'album', 'spotify_popularity', 'genius_url', 'genius_tags'])
    for i in album_ids:
        for j in spotify_get(f'/albums/{i}/tracks', {})['items']:
            spotify_id = j['id']
            track = spotify_get(f'/tracks/{spotify_id}', {})
            name = track['name']
            artists = [x['name'] for x in track['artists']]
            album = track['album']['name']
            spotify_popularity = track['popularity']
            genius_search = genius_get('/search', {'q': name+' '+' '.join(artists)})['response']['hits']
            if 'pageviews' in genius_search[0]['result']['stats']:
                genius_url = genius_search[0]['result']['url']
                genius_tags = genius_get_tags(genius_url)
            else:
                genius_url = np.NAN
                genius_tags = np.NAN
            df.loc[len(df)] = [spotify_id, name, artists, album, spotify_popularity, genius_url, genius_tags]
    return df

Linear Regression¶

Now, we will use linear regression to determine whether there is a predictive relationship between genius_tags_count and spotify_popularity.

In [11]:
from sklearn.linear_model import LinearRegression

model = LinearRegression().fit([[x] for x in df3['genius_tags_count']], [[y] for y in df3['spotify_popularity']])

fig, ax = plt.subplots(figsize=(12,8))
ax.plot([[x] for x in df3['genius_tags_count'].unique()], [y for [y] in model.predict([[x] for x in df3['genius_tags_count'].unique()])])
plt.scatter(x=df3['genius_tags_count'], y=df3['spotify_popularity'], alpha=.2, s=80)
plt.title('Popularity by Number of Genre Tags of Bladee Songs')
plt.ylabel('Popularity')
plt.xlabel('Number of Genre Tags')
plt.show()

It seems there is a positive relationship between the two variables, meaning as the number of tags goes up, the popularity of the song is likely to go up.

4. Conclusion¶

It seems that the more popular genres are closer to Bladee's current artistic direction. For example, Amygdala, a recent release, while not considered in this report, was tagged on Genius as Dance-Pop. Bladee's popularity has only gone up over time, so it makes sense that his more popular material is closer to his current sound. It is interesting to see that songs that draw from more genres are more popular. This might reveal something about what Bladee's fans, or music fans in general, value today: more eclectic music drawing from broad influences.

It would be interesting to compare how certain combinations of genres affect a song's popularity. It would also be interesting to use data from the website Every Noise at Once, where relationships between genres are represented visually.

This is an analysis that will likely change over time, so it is important to document these data as a snapshot in time, preserved for appreciation and comparison later.
